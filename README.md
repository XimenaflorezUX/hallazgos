# UX Research Canvas — Magneto Platform

Dashboard estático de hallazgos UX para los canales **B2C** (candidatos) y **B2B** (reclutadores) de Magneto Platform.

## Uso

Abre `index.html` en el navegador o sirve la carpeta con cualquier servidor estático.

## Stack

- HTML + [Tailwind CSS](https://tailwindcss.com/) (CDN)
- [Lucide](https://lucide.dev/) icons (CDN)

## Contenido

- Métricas B2C y B2B (CSAT, CES, muestra)
- Tarjetas de dolor UX con filtros por categoría
- Hojas de ruta por canal
